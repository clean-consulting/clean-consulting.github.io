# How to reflect the change on development server
1. Login to [PythonAnywhere](https://www.pythonanywhere.com)
2. Go to dashboard
3. Open the bash console
4. ```cd oneone.pythonanywhere.com```
5. ```git pull```
6. Go to Web
7. Click reload oneone.pythonanywhere.com
